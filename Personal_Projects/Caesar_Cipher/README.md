# Caesar Cipher

## Description
A Python program to encrypt and decrypt messages using the Caesar Cipher technique. 
The user can encode or decode a message by providing a shift number.

## Features
- Encrypt messages
- Decrypt messages
- Handles non-alphabetic characters
- Simple command-line interface

## Skills Learned
- Python lists and loops
- Functions
- Conditionals
- User input handling

## How to run
```bash
python main.py

## Example
Type 'encode' to encrypt and 'decode' to decrypt:

Hello World
