# Hangman Game – States of India Edition

## Description
A Python game where the player guesses the names of Indian states. 
The game shows hangman graphics for wrong guesses and counts lives.

## Features
- Random state selection
- Visual hangman stages
- Tracks guessed letters
- Replay option

## Skills Learned
- Python lists and loops
- Random module
- Error handling
- String manipulation

## How to Run
```bash
python main.py
