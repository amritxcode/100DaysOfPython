# Notes Manager

## Description
A command-line Notes Manager to add, view, and delete notes. 
Counts words and lines for each note.

## Features
- Add notes
- View notes with numbering
- Delete specific notes
- Count words and lines
- Handles missing file errors

## Skills Learned
- Python file handling
- Loops and conditionals
- Error handling (try-except)
- User input handling

```bash
python main.py
