# 🧮 Word Counter Project

## 📘 Overview  
This project is a simple **Word Counter** built in Python, created as part of my ongoing learning journey to strengthen core programming logic and text manipulation skills.  
The program takes a user-input string and counts the total number of words, demonstrating concepts like string handling, loops, and basic algorithmic thinking.

---

## 🧠 Key Concepts Practiced
- String manipulation  
- Input handling  
- Loops (`for`, `while`)  
- Conditional statements  
- Basic data structures and logic design  

---

## ⚙️ How It Works
1. The program takes an input sentence from the user.  
2. It splits the sentence into words using Python’s built-in string functions.  
3. Counts the total words and displays the result neatly.
